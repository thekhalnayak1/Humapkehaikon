Frequently Asked Questions

How do I add Radio Bots to my server?

You can add Radio Bots to your Discord server by clicking on the invite button. You must have a role with the “Administrator” or the “Manage Server” permission to be able to add to your Discord Server.

How do I play music with the bot?

Adding music to the playlist is pretty simple. To start playing a track, use the /play command. Here are two examples:
/play https://open.spotify.com/track/62PaSfnXSMyLshYJrlTuL3
/play hello

How to set the bot 24/7 in VC?

First you have to join the voice channel there you can set the bot 24/7 then you can simply set the bot 24/7 by using /24/7.
